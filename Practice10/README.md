# Напишите программу "Дневник погоды":
## 1. Каждый день записывает:
- Дату
- Температуру
- Погоду (солнечно, облачно, дождь)
## 2. Может выводить статистику:
- Средняя температура за неделю
- Количество солнечных дней
- Самая высокая / низкая температура

## Пример записи:
Дата: 2024-01-15
Температура: 25
Погода: солнечно
