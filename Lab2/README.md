# Lab2

Сравнение производительности разных коллекций в .NET.

Тестируются List, LinkedList, Queue, Stack и ImmutableList. Для каждой коллекции измеряется время выполнения операций добавления, удаления, поиска и доступа по индексу.

## Запуск

```bash
dotnet restore
dotnet build
dotnet run
```

Для запуска тестов:

```bash
cd Lab2.Tests
dotnet test
```

## Параметры

- Размер коллекций: 100000 элементов
- Итераций измерения: 5
- Прогрев: 3 итерации

Результаты выводятся в консоль в виде таблиц.
