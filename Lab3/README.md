# Лабораторная работа 3

## Тема
Реализация коллекций и интерфейсов в C# (.NET)

## Цель
Познакомиться с интерфейсами коллекций в C#, реализовать простые коллекции и понять их контракт и функциональность.

## Задание

### 1. SimpleList
Реализация динамического массива, который:
- Реализует интерфейсы `IEnumerable`, `ICollection`, `IList`
- Внутренне использует динамический массив (без готовых коллекций)
- Поддерживает: добавление, получение/установку по индексу, удаление, поиск индекса
- Методы: `Count`, `Add`, `Remove`, `Clear`, `Contains`, `IndexOf`, `Insert`, `RemoveAt`

### 2. SimpleDictionary<TKey, TValue>
Реализация хеш-таблицы, которая:
- Реализует `IDictionary<TKey, TValue>` и `IReadOnlyDictionary<TKey, TValue>`
- Использует собственный массив с цепочками для разрешения коллизий
- Поддерживает добавление, удаление и поиск по ключу
- Обеспечивает перебор через `IEnumerable<KeyValuePair<TKey, TValue>>`

### 3. BidirectionalLinkedList
Реализация двунаправленного связного списка:
- Реализует интерфейс `IList`
- Каждый узел содержит ссылки на предыдущий и следующий элементы
- Поддерживает все операции `IList` с оптимизацией доступа по индексу
